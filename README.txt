Důležité:
Tento balík je připravený jako vícestránková kopie poslední jednostránkové verze.
Ponechává stejné cesty k obrázkům jako původní web:
- images/MukapesLOGO.png
- images/tricko-cervene.png
- images/ksiltovky.jpeg
- images/galerie-2022/*

Pokud web nahráváš přes Netlify/FTP přes současnou verzi, nemaž původní složku images.
Nahraj obsah této složky do kořenové složky webu a přepiš HTML soubory.

Formuláře:
- Rezervace merche je na stránce merch.html
- Odběr novinek je na stránce kontakt.html
- Po odeslání formuláře vede přesměrování na dekujeme.html

Kontrola po nahrání:
1. Otevři index.html
2. Projdi hlavní menu
3. Zkontroluj galerii 2022
4. Zkontroluj Netlify Forms: rezervace-merche a novinky
